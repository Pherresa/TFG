\paragraph {Descripci\'on:}
El proyecto consistir\'a en el dise\~no y desarrollo de un entrono gr\'afico que
permita el estudio y comprensi\'on de algunos de los algoritmos cl\'asicos
estudiados en la asignatura de Sistemas Operativos:
\begin {itemize}
\item Algoritmos de planificaci\'on de CPU.
\item Algoritmos de planificaci\'on de memoria principal.
\item Algoritmos de planificaci\'on de acceso a disco magn\'etico.
\item Algoritmos de remplazo de pagina.
\item Gesti\'on de un buffer cache (UNIX).
\end {itemize}
\paragraph {Calendario:}
\begin {itemize}
\item Estudio de la librer\'\i a GTK+
\item Diseño de la aplicaci\'on
\item Implementaci\'on de las distintas partes de la aplicaci\'on:
	\begin {enumerate}
	\item Modulo de CPU.
	\item Interface con el resto de m\'odulos.
	\item Modulo de Memoria.
	\item Modulo disco.
	\item Buffer Cache.
	\end {enumerate}
\item Realizaci\'on de la memoria.
\end {itemize}
\paragraph {Algoritmos a implementar}
\begin {itemize}
\item Planificaci\'on de CPU.
	\begin {itemize}
	\item FCFS
	\item Round Robin
	\item SPN
	\item SRT
	\item HRRN
	\item Opcional: Unix y Windows NT
	\item C\'alculos de productividad:
		\begin {itemize}
		\item Tiempos de respuesta
		\item Tiempos de retorno
		\item ...
		\end {itemize}
	\end {itemize}
\item Planificaci\'on de memoria principal.
	\begin {itemize}
	\item Partici\'on fija
	\item Partici\'on din\'amica
	\item Visualizaci\'on gr\'afica de la figuras
	\item C\'alculos de fragmentaci\'on interna y externa
	\end {itemize}
\item Reemplazo de pagina.
	\begin {itemize}
	\item OPT
	\item LRU
	\item FIFO
	\item CLOCK
	\item Calculo de n\'umeros de fallos de pagina
	\end {itemize}
\item Acceso a disco magn\'etico.
	\begin {itemize}
	\item FIFO
	\item SSTF
	\item SCAN
	\item C-SCAN
	\end {itemize}
\item Gesti\'on de un buffer cache.
	\begin {itemize}
	\item Visualizaci\'on de cola de libres
	\item Visualizaci\'on de las colas hash
	\item Visualizaci\'on de escenarios para la recuperaci\'on de un buffer
	\end {itemize}
\end {itemize}
